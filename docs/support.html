Alpha Support

For build problems, crashes, sync issues, or suspicious behavior.

Primary Support

  • Email: support@yourcraft.gg
  • Bug reports: Report directly in-app during alpha.
  • Security reports: security@yourcraft.gg

What to Include in Bug Reports

  • App version and platform (example: 0.1.1, macOS aarch64).
  • Exact reproduction steps.
  • Expected result vs actual result.
  • Screenshot/log snippets if available.

Severity Routing

  • P0: data loss, auth break, startup crash, security exposure.
  • P1: major feature broken without workaround.
  • P2: degraded behavior with workaround.
  • P3: minor UI/polish issues.

Status

Product bugs and UX issues are reported directly in-app during alpha.
