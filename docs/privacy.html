Privacy Policy

Last updated: February 25, 2026

Scope

This policy applies to the Yourcraft alpha desktop application, website, and related support channels.

Data We Collect

  • Account information: username, profile metadata, auth identifiers.
  • Gameplay data: characters, progression, PvP records, quests, and rewards.
  • Operational metadata: timestamps, session summaries, and service diagnostics.

Data We Do Not Intend to Store in Cloud

  • Local terminal transcript content from your coding sessions.
  • Your local filesystem contents.
  • Private API keys used in your local coding environment.

How We Use Data

  • Deliver core product features and progression systems.
  • Operate updates, reliability monitoring, and support workflows.
  • Investigate bugs and abuse reports during alpha.

Security

We apply reasonable safeguards for alpha operations, including authenticated API access, key management, and restricted administrative privileges. Alpha systems may still evolve rapidly.

Contact

For privacy requests or concerns, contact security@yourcraft.gg.
